# Игра Память (Memory Game)

1. Клонируйте репозиторий:

   git clone "url"
   cd memory-game

2. Установите зависимости:

   npm install

3. Запустите проект в режиме разработки:

   npm run dev

4. Откройте браузер:
   Перейдите по адресу [http://localhost:5173](http://localhost:5173)

Технологии

- Vue.js 3
- Vite
- CSS3
- JavaScript ES6
